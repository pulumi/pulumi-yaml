## v1.33.0 - 2026-05-04

### Improvements

- [runtime] Support the pulumiresourcename and pulumiresourcetype functions [#1060](https://github.com/pulumi/pulumi-yaml/pull/1060)

- [runtime] Support the `fn:sha1` built-in function [#1062](https://github.com/pulumi/pulumi-yaml/pull/1062)

- [runtime] Support `filebase64` and `filebase64sha256` intrinsic functions [#1063](https://github.com/pulumi/pulumi-yaml/pull/1063)

- [runtime] Support the `singleOrNone` and `length` intrinsic functions [#1069](https://github.com/pulumi/pulumi-yaml/pull/1069)

### Bug Fixes

- [runtime] YAML components can now be instantiated multiple times in a single stack. Child resource names are prefixed with the component instance name (e.g. `cp1-res`); aliases to the un-prefixed name are emitted so existing single-instance stacks migrate without replacement. [#957](https://github.com/pulumi/pulumi-yaml/pull/957)
