## v1.34.0 - 2026-05-13

### Improvements

- [runtime] Support `fn::length` on strings, returning the grapheme cluster count to match PCL's `length()` builtin [#1085](https://github.com/pulumi/pulumi-yaml/pull/1085)

- [runtime] Support `fn::unsecret` to unwrap a secret value, matching PCL's `unsecret()` builtin [#1086](https://github.com/pulumi/pulumi-yaml/pull/1086)

- [runtime] Allow `customTimeouts` create/update/delete values to be expressions (e.g. config references), not just string literals [#1087](https://github.com/pulumi/pulumi-yaml/pull/1087)

### Bug Fixes

- [runtime] Allow `fn::join` to accept the result of `fn::split` by returning a generic list rather than a string-typed slice [#1083](https://github.com/pulumi/pulumi-yaml/pull/1083)
